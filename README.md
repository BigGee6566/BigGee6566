## Hi there
This is my personal developer portfolio website built using **HTML and CSS**.  
It showcases my skills, projects, certificates, and contact information in a clean, modern, and responsive design.

---

##  About the Portfolio

This portfolio was created to:
- Present my **projects** and the tools used
- Display my **technical skills**
- Showcase my **certificates and achievements**
- Provide an easy way to **contact me**

The design focuses on simplicity, readability, and a professional layout inspired by modern UI principles.

---

## Technologies Used

- **HTML5** – Page structure and content  
- **CSS3** – Styling, layout, responsiveness, and animations  
- **Font Awesome** – Social media icons  
- **Devicon** – Technology skill icons  

>  No JavaScript was used for functionality to keep the project lightweight and easy to understand.

---

## Features

- Responsive layout (desktop & mobile)
- Hero section with profile image
- Skills section with technology icons
- Projects section with descriptions and tool badges
- Certificates gallery (6 certificates)
- Contact form UI
- Clean dark-themed design with accent colors
